SETUP STEPS:
============

Step 1: Create Folders
-----------------------
mkdir umkm-forecasting-app
cd umkm-forecasting-app
mkdir backend frontend data

Step 2: Add Files
-----------------
Copy all provided files to their respective folders:

backend/
  - Copy main.py content
  - Copy requirements.txt content

frontend/
  - Copy index.html content
  - Copy app.js content

Root directory:
  - Copy README.md content
  - Copy SETUP_GUIDE.md content
  - Copy start_app.bat content (Windows)
  - Copy start_app.sh content (Mac/Linux)

Step 3: Make Scripts Executable (Mac/Linux only)
------------------------------------------------
chmod +x start_app.sh

Step 4: Add Your Data
----------------------
Place your catatan_umkm.csv file in the data/ folder

Step 5: Run the App
-------------------
Windows:
  Double-click start_app.bat
  OR
  Open cmd and run: start_app.bat

Mac/Linux:
  Open terminal and run: ./start_app.sh
  OR
  bash start_app.sh

TROUBLESHOOTING:
================

Problem: "Port already in use"
Solution: 
  Windows: netstat -ano | findstr :8000
           taskkill /PID <PID> /F
  Mac/Linux: lsof -ti:8000 | xargs kill -9

Problem: "Module not found"
Solution: Make sure venv is activated
          pip install -r requirements.txt

Problem: "CORS error"
Solution: Use HTTP server, not direct file opening
          Don't open index.html directly

Problem: "File upload fails"
Solution: Check CSV column names match exactly
          Ensure dates can be parsed
